# Email Triage AI

An AI-powered email assistant that classifies emails, assigns priority, and generates responses using Llama 3.1.

## Features
- Classifies emails: spam / important / normal
- Assigns priority: low / medium / high
- Generates short context-aware replies

## Setup
Set HF_TOKEN as a secret in Space settings.

## Run
App starts automatically on launch.
